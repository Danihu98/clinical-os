# Doing CBT

**Doing CBT: A Comprehensive Guide to Working with Behaviors, Thoughts, and Emotions** es un libro escrito por David F. Tolin, publicado por Guilford Press. Es una de las referencias principales utilizadas en este espacio clínico.

El modelo de formulación propuesto por Tolin integra componentes conductuales, cognitivos y emocionales en un esquema visual que permite mapear las relaciones entre los diferentes elementos que mantienen las dificultades del paciente. Este modelo sirve como base para la plantilla de Canvas principal utilizada en este sistema.

### Relevancia

Lo que hace valioso el modelo de Tolin es su integración de múltiples componentes en una [[Formulaciones clínicas|formulación]] visual y comprehensiva. En lugar de enfocarse exclusivamente en cogniciones (como el modelo de Beck) o en conductas (como el análisis funcional puro), el modelo propone una visión que captura las interacciones entre todos estos elementos, permitiendo una comprensión más completa del caso y facilitando la toma de decisiones sobre dónde intervenir.
